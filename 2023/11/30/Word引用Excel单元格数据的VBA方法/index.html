Word引用Excel单元格数据的VBA方法

​ 日报类工作,经常会遇到从excel复制数据到word文档的情况,利用word的书签及宏功能,本文尝试用粗糙的VBA编码,大大降低这种工作的繁琐程度。

新建一个控制按钮关联上下面的代码就可以了,原理都在里面,可以根据自己的情况进行修改:

��，用逗号分隔
Dim dataArray As String
For i = LBound(excelData1, 1) To UBound(excelData1, 1)
dataArray = dataArray & excelData1(i, 1) & vbCrLf
Next i

' 将数据更新到 Word 文档的指定位置
wordDoc.Bookmarks("当日发生业务").Range.Text = dataArray

‘————————————————————————————-
‘ 关闭 Excel 工作簿
excelWorkbook.Close
‘ 释放对象
Set excelSheet = Nothing
Set excelWorkbook = Nothing
Set bookmarkRange = Nothing
Set excelRange = Nothing
Set excelApp = Nothing
Set wordDoc = Nothing

' 关闭 Excel 应用程序
If Not excelApp Is Nothing Then
    excelApp.Quit
    Set excelApp = Nothing
End If

' 结束 Excel 进程
On Error Resume Next
Dim excelProcess As Object
For Each excelProcess In GetObject("winmgmts:").ExecQuery("Select * from Win32_Process Where Name = 'EXCEL.EXE'")
    excelProcess.Terminate
Next
On Error GoTo 0

End Sub
